# Stage 1 精講稿 — Estimation 與 Sampling Distribution

一句話主旨:**estimator 是隨機變數;它的分布叫 sampling distribution;它的標準差叫
standard error;對 sample mean 而言 SE = σ/√n。** 文章所有「單一數字」都該被你心裡
自動套上這條 error bar。

---

## 1. LLN 與 CLT:收斂與速率(M1.1)

設 `X₁,…,Xₙ` i.i.d.,均值 `μ`、變異 `σ²`。樣本均值 `X̄ₙ = (1/n)ΣXᵢ`。

- **大數法則(LLN)**:`X̄ₙ → μ`(n→∞)。估計量會收斂到真值——但**沒說多快**。
- **中央極限定理(CLT)**:
  ```
  √n (X̄ₙ - μ)  --d-->  N(0, σ²)
  ```
  等價地 `X̄ₙ ≈ N(μ, σ²/n)`。**速率是 √n**:要把不確定性砍半,樣本要變 4 倍。

這條 √n 是整套故事的引擎。n=9 → √n=3;n=100 → √n=10。從 9 到 100,精度只
進步 `10/3 ≈ 3.3` 倍——而 9 本身的起點已經爛到 SE 跟訊號同數量級。

---

## 2. Point estimation 的四個性質(M1.2)

評價一個估計量 `θ̂`:

| 性質 | 定義 | 白話 |
|---|---|---|
| **Bias** | `E[θ̂] - θ` | 平均而言有沒有系統性偏掉 |
| **Variance** | `Var(θ̂)` | 換一個樣本,估計值會跳多少 |
| **Consistency** | `θ̂ →ᵖ θ` | n→∞ 會不會收斂到真值 |
| **Efficiency** | variance 是否達到下界(Cramér–Rao) | 在無偏估計裡是不是最省資訊 |

關鍵:**unbiased 不代表可靠。** `X̄` 對 μ 是無偏的,但 n=9 時它的 variance 巨大——
單看一次樣本,你拿到的那個數可能離 μ 很遠。文章的 +4.7% 也許是「無偏」的,但那無關
緊要;要命的是它的 **variance**。

均方誤差把兩者合一:`MSE(θ̂) = Bias² + Var`。

---

## 3. Maximum Likelihood(M1.3)

likelihood 是「在參數 θ 下,觀察到這份資料的機率」當成 θ 的函數:

```
L(θ) = Π f(xᵢ; θ),     ℓ(θ) = log L(θ) = Σ log f(xᵢ; θ)
θ̂_MLE = argmax_θ ℓ(θ)
```

直覺:挑「最能生出我手上這份資料」的參數。常態母體下 `μ̂_MLE = X̄`。
MLE 在正則條件下 consistent、漸近常態、漸近 efficient:

```
√n (θ̂_MLE - θ)  --d-->  N(0, I(θ)⁻¹)
```

`I(θ)` 是 Fisher information。重點對判讀的用途:**MLE 的精度也是 O(1/√n)**——
小樣本一樣救不了。

---

## 4. 核心:Sampling distribution、SE = σ/√n(M1.4)

### 4.1 sampling distribution 是什麼

想像可以反覆從母體抽 n 個、每次算一次 `X̄`。這些 `X̄` 自己構成一個分布,就是
**sampling distribution of the estimator**。它**不是**資料的分布,是「估計值」的分布。

- 中心:`E[X̄] = μ`(無偏)。
- 散度:`Var(X̄) = σ²/n`。

### 4.2 手推 Var(X̄) = σ²/n

```
Var(X̄) = Var( (1/n) Σ Xᵢ )
        = (1/n²) Var( Σ Xᵢ )
        = (1/n²) Σ Var(Xᵢ)        (i.i.d. ⟹ 共變數為 0)
        = (1/n²) · n σ²
        = σ² / n
```

取根號:**`SE(X̄) = σ/√n`**(σ 未知時用樣本標準差 s 代入,得 `ŜE = s/√n`)。
`√n` 的來源就是上面那步「n 個獨立項的變異相加、再被 n² 除」——**獨立性是關鍵**;
若資料有正自相關(金融報酬常見),`Σ Var` 那步會多出共變數項,真正的 SE 會比
`s/√n` **更大**(這是 Stage 5「effective sample size」的伏筆)。

### 4.3 n=9 為何吞掉一切

把數字代進去。設年報酬 σ ≈ 17%:

| n | SE = σ/√n | 95% 區間半寬 ≈ 2·SE |
|---|---|---|
| 9 | 17%/3 = **5.7%** | ±11.3% |
| 100 | 17%/10 = **1.7%** | ±3.4% |

n=9 時,一個「平均 +4.7%」的 95% 區間大約是 `+4.7% ± 11%`,從 −6% 蓋到 +16%——
它跟「+8.5% 的 buy-and-hold」根本分不開。**這不是運氣不好,是 σ/√n 的算術。**

從 9 到 100,SE 縮小倍數 = `√(100/9) = 10/3 ≈ 3.33`,所以區間縮成約 **1/3**。
這就是 gate 要你解釋的那句話。

### 4.4 SE vs SD,別搞混

- **SD(標準差)**:單一觀測值的離散程度,`σ`。描述**資料**。
- **SE(標準誤)**:估計量的離散程度,`σ/√n`。描述**估計值**。

財經評論最常見的偷換:報一個 SD 很大的東西的平均值,卻當成那個平均值很確定。
你要永遠分清「資料多散」與「我對平均的估計多準」是兩件事。

---

## 5. 一頁回顧

| 概念 | 式子 | 判讀用途 |
|---|---|---|
| CLT | `X̄ ≈ N(μ, σ²/n)` | 任何平均都近似常態,散度 σ/√n |
| SE of mean | `SE = σ/√n` | 給任何「單一平均」配 error bar |
| 收斂速率 | √n | 精度翻倍要 4 倍樣本;n=9→100 只快 3.3× |
| unbiased ≠ 可靠 | `MSE=Bias²+Var` | +4.7% 可能無偏但 variance 巨大 |
| 獨立性假設 | `Var(ΣX)=ΣVar` | 報酬有自相關 ⟹ 真 SE 更大(Stage 5) |

下一站(Stage 2):有了 SE,怎麼把它變成**正確的**信賴區間(小樣本/比例不能用
Wald),以及為什麼 n=9 幾乎沒有 power 去拒絕任何東西。
