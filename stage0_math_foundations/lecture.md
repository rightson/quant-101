# Stage 0 精講稿 — 數學地基

目標讀者:會基本代數、想把「條件 vs 無條件」這件事弄到能徒手推導的程度。
全程用一個貫穿例子:**X = 今天是否創一年新高,Y = 之後 N 日的報酬。**

---

## 1. Integration 就是「對 density 求期望」(M0.1)

連續隨機變數 `X` 的機率密度 `f(x)` 滿足兩件事:

- 非負:`f(x) ≥ 0`
- 全積分為 1:`∫ f(x) dx = 1`

期望值就是用 density 當權重對 x 做加權平均:

```
E[X] = ∫ x f(x) dx
E[g(X)] = ∫ g(x) f(x) dx          (LOTUS:不必先求 g(X) 的分布)
```

**為什麼這對判讀重要**:當有人說「平均報酬是 +4.7%」,他報的是
`∫ x f(x) dx` 這個*單一數字*。但這個數字背後是整條 `f`。一條右偏、帶左尾的
`f`(先漲為主、偶爾崩)可以給出漂亮的正平均,卻完全不保證「下一次不會踩到左尾」。
Integration 的視角讓你永遠記得:**平均是把整條分布壓成一個點,壓掉的資訊正是風險所在。**

Variance 同理:`Var(X) = E[(X-μ)²] = ∫ (x-μ)² f(x) dx`。它量的是「density 攤多開」。

## 2. Projection 就是 OLS 的幾何(M0.2)

把資料想成向量空間裡的向量。給定 `Y ∈ ℝⁿ` 與設計矩陣 `X`(各 column 是一個解釋
變數),最小平方:

```
β̂ = argmin_β ‖Y - Xβ‖²
```

幾何上,`Xβ` 走遍 `X` 的 column 張成的子空間 `col(X)`。讓 `‖Y - Xβ‖²` 最小的
`Xβ̂`,正是 `Y` 在 `col(X)` 上的**正交投影**。一階條件:

```
Xᵀ(Y - Xβ̂) = 0   ⟹   β̂ = (XᵀX)⁻¹ XᵀY
```

殘差 `Y - Xβ̂` 與 `col(X)` 正交。投影矩陣 `H = X(XᵀX)⁻¹Xᵀ`(hat matrix)滿足
`H² = H`(投影兩次等於投影一次)、`Hᵀ = H`。

**為什麼這對判讀重要**:Stage 6 會把「創新高後 +4.7%」拆成
`市場 drift + 動能因子` 兩塊——那就是把報酬投影到「已知因子」張成的子空間,看殘差
還剩多少。如果殘差≈0,那「需要主力洗盤才能解釋」的東西根本不存在。投影是「拆解」
的數學。

## 3. 隨機變數、分布、期望、變異(M0.3)

- **Random variable**:把隨機結果對應到數的函數。`Y` = 某天之後 20 日的報酬。
- **CDF** `F(y) = P(Y ≤ y)`;**PDF/PMF** 是它的密度/質量。
- **Expectation** `E[Y]`:長期平均。**線性**:`E[aY+bZ] = aE[Y]+bE[Z]`(永遠成立,
  即使 Y、Z 相關)。
- **Variance** `Var(Y) = E[Y²] - (E[Y])²`;`Var(aY) = a²Var(Y)`。

一個之後很關鍵的恆等式(variance decomposition,Stage 5/9 會用):

```
Var(Y) = E[Var(Y|X)] + Var(E[Y|X])
```

「總變異 = 組內平均變異 + 組間均值的變異」。

## 4. 核心:Joint / Marginal / Conditional(M0.4)

這一節是整站的重心,也是文章第五錯的數學根。

### 4.1 三者的定義

對兩個隨機變數 `X, Y`:

- **Joint**:`f(x, y)` —— 同時描述兩者。
- **Marginal**:把另一個積掉。`f_X(x) = ∫ f(x,y) dy`。
- **Conditional**:固定 `X=x` 後 `Y` 的分布。
  ```
  f_{Y|X}(y | x) = f(x, y) / f_X(x)        (要求 f_X(x) > 0)
  ```

口訣:**conditional = joint ÷ marginal**。徒手推導就是這一行——把 joint 沿著
`X=x` 切一刀,再除以那一刀的總質量讓它重新積分成 1。

### 4.2 條件期望 `E[Y|X]`

```
E[Y | X=x] = ∫ y · f_{Y|X}(y|x) dy
```

注意 `E[Y|X]` 是 **X 的函數**(對每個 x 給一個數),它本身是隨機變數。

**Law of Total Expectation(全期望公式)**:

```
E[Y] = E[ E[Y|X] ]
```

白話:無條件平均 = 對「各條件平均」再按 X 的分布加權平均。離散版:

```
E[Y] = Σ_x  E[Y|X=x] · P(X=x)
```

### 4.3 為什麼 `E[Y | X=創新高] ≠ E[Y]`

設 `X = 1{今天創一年新高}`。把所有交易日的「未來 N 日報酬」`Y` 看成一個母體。

- `E[Y]` = **無條件**平均報酬:把所有日子混在一起。
- `E[Y | X=1]` = 只在「創新高那些日子」裡算的平均報酬。

這兩個一般不相等。差距 `E[Y|X=1] - E[Y]` 的大小,完全由「創新高」這件事和未來報酬
的相關程度決定。而「創新高」幾乎定義上就是「近期報酬為正且夠大」——它是
**positive momentum 的指示函數**。所以:

> **對「創新高」做 conditioning,本質上就是對「近期動能為正」做 conditioning。**
> `E[Y|創新高]` 之所以跟 `E[Y]` 不同,是因為我們只挑了母體裡動能為正的那一塊;
> 若動能本身有前瞻性(Stage 6 會說它確實有,這叫 momentum premium),那
> `E[Y|創新高] > E[Y]` 是**機械性的結果**,不需要任何「主力」或「洗盤」的故事。

這就是文章「第五錯」:把一個 *conditioning 造成的選擇效果*,誤讀成一個*需要敘事解釋的
現象*。你在 Stage 0 學會的這一行 `conditional = joint ÷ marginal`,就是拆穿它的工具。

### 4.4 獨立 = conditioning 沒用

若 `X ⟂ Y`(獨立),則 `f_{Y|X}(y|x) = f_Y(y)`,於是 `E[Y|X] = E[Y]`——
**這時候 conditioning 完全不改變期望**。所以「創新高後報酬更高」要成立,
前提就是「創新高」與「未來報酬」**不獨立**。檢查一個型態是不是 signal,某種意義上
就是檢查這個條件期望差是不是真的、還是抽樣雜訊(這正是 Stage 1–2 的工作)。

## 5. 一頁回顧

| 概念 | 式子 | 判讀上的用途 |
|---|---|---|
| 期望 = 對 density 加權 | `E[g(X)]=∫ g f dx` | 平均把分布壓成一點,壓掉的是風險 |
| OLS = 投影 | `β̂=(XᵀX)⁻¹XᵀY` | 把報酬拆成「因子能解釋的」+「殘差」 |
| conditional = joint÷marginal | `f_{Y|X}=f(x,y)/f_X(x)` | conditioning 的定義 |
| 全期望 | `E[Y]=E[E[Y|X]]` | 無條件 = 條件的加權平均 |
| 獨立 ⟹ 條件無效 | `X⟂Y ⟹ E[Y|X]=E[Y]` | signal 的前提是「不獨立」 |
